# Top-K-words-Algorithm
Program that outputs the top k words from a text file and displays them
The paradigm is functional programming thus the program will use the criteria for functional programming such as 
no mutation. It takes an input list and stopwords that are to be ignored and displays the top k words as specifed by the arguments passed 
the program is executed as follows python3 keywords.py input=input.txt;k=NUMBER;mostfrequent=Y/N;uppercase=Y/N;ouput=topkword.txt
and requires the stopwords.txt to be in the directory
